# What?
Python + Flask implementation of [todobackend.com](https://www.todobackend.com/)

# Prerequisites

- Python 3.6
- pipenv

# Running

### Install required modules

`pipenv install`

### Run the application

`pipenv run python hello.py`

### Run the tests

Go [here](https://www.todobackend.com/specs/index.html?http://localhost:5000)
